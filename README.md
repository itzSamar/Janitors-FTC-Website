# The Janitors FTC Website

Official website for **The Janitors**, a FIRST Tech Challenge robotics team.

## Pages

- **Home** — Hero, mission, and call to action
- **About** — Team story, values, and roster (placeholders)
- **Outreach** — Community initiatives and impact log
- **Robots** — Current season robot and design highlights
- **Apply** — Join the team

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build for production

```bash
npm run build
npm run preview
```

The `dist/` folder can be deployed to GitHub Pages, Netlify, Vercel, or any static host.

## Customization

Content placeholders are marked throughout the site. Update team email, social links, photos, roster, and application form URL as your season progresses.
